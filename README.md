
# url_shortner

Ruby:
Application used ruby-2.4.0 version

Rails:

This applicaiton uses 5.1.3 rails version

To set up a development environment:

    -> git clone https://github.com/MiteshKothari/url_shortner.git
    -> cd url_shortner
    -> bundle install
    -> rake db:create && rake db:migrate
    -> rails server
